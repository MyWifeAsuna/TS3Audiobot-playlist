# TS3Audiobot-playlist

 把自己的歌单转换成teamspeak3的歌单



## 说明

- 目前只支持网易云，且不支持vip
- 输入歌单ID自动爬取，不支持私人歌单
- 硬爬歌单转换成TS3Audiobot所使用的格式，存放在 `./bot/botname/playlists` 目录下，也可以放在自己指定的位置

生成好的文件存放在`data`目录下  

[下载地址](https://github.com/MyWifeAsuna/TS3Audiobot-playlist/releases)
